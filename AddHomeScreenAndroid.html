Install 2026 C2C Rides

Follow one of the options below to install this site to your Android device.

Option 1: Using Chrome (most common way)

  1. Open Chrome on your Android phone.
  2. Go to https://c2c2026.netlify.app.
  3. Tap the 3 dots menu (⋮) in the top right corner.
  4. Choose Add to Home screen or Install app (wording varies slightly).
  5. You'll see a preview icon and name — you can edit the name if you like.
  6. Tap Add → then Add automatically (or drag the icon to a spot manually).
  7. Select an icon or an image from your photo library.
  8. Done ✅ You now have a home screen icon that opens the website in a standalone window, just like a native app.

Option 2: Using Samsung Internet

  1. Open Samsung internet browser.
  2. Go to https://c2c2026.netlify.app.
  3. Tap the menu (≡) → Add page to → Home screen.
  4. Confirm and name the shortcut.
  5. Done ✅ You now have a home screen icon that opens the website in a standalone window, just like a native app.

Option 3: Using Firefox

  1. Open Firefox.
  2. Go to https://c2c2026.netlify.app.
  3. Tap the three dots (⋮) → Add to Home screen.
  4. Confirm and the icon appears on your home screen.
  5. Done ✅ You now have a home screen icon that opens the website in a standalone window, just like a native app.
